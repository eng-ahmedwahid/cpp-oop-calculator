# C++ OOP Calculator

Simple calculator project built using C++ and Object-Oriented Programming (OOP).

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Power function
- Cancel last operation
- Clear calculator
- Tracks last operation and result

## Concepts Used
- Classes and Objects
- Encapsulation
- OOP design
- Functions
- Basic input validation
